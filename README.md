# Node TypeScript API
API using Node TypeScript Jest using 3rd party OpenWeatherMap API to retrieve weather info

## Dependencies
- node@8.4.0
- npm@5.6.0

## Stack
- Dev
  - Node
  - TypeScript
- Test
  - Jest

## Execute the API
```
npm install
npm start
```

## Structure
- Controllers (Business Logic)
- Presenters (Folder for each API)
- Routers (File for each API)

## Test considerations

### Execute
```
npm test
```
